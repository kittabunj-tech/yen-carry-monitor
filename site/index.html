Methodology ▾

CUSI คำนวณอย่างไร

รวม 9 components เป็น composite z-score ถ่วงน้ำหนัก แล้วแปลงเป็นสเกล 0–100: CUSI = 50 + (weighted_z / 3) × 50 (clip 0–100) โดย z-score ใช้หน้าต่าง 252 วันทำการ clip ที่ ±3 และทุก component ถูกปรับทิศทางให้ ค่าบวก = เสี่ยงมากขึ้น เสมอ

ระดับ Alert

    ข้อควรรู้

    • ไม่มี lookahead — ค่าที่วัน t ใช้ข้อมูลถึงวัน t เท่านั้น ข้อมูล COT (วัดวันอังคาร ประกาศบ่ายวันศุกร์) ถูก forward-fill จาก วันที่ประกาศ ไม่ใช่วันที่วัด
    • Re-normalization — วันไหน component ขาด (เช่น MOVE ค้าง) ระบบจะเฉลี่ยเฉพาะน้ำหนักที่มีข้อมูลจริง ไม่ใช่แทนค่าด้วย z = 0
    • JGB 10Y มาจาก MOF ญี่ปุ่น (ทางการ) — ถ้า MOF ล่ม จะสลับไปใช้ ETF proxy และติดธงเตือนไว้บนหัวเว็บ
    • Real-time vs Computed — widget แถวที่ 2 เป็นราคาสดจาก TradingView ส่วน CUSI และตารางด้านล่างมาจาก pipeline ที่รันเป็นรอบ จึงอาจต่างกันได้ตามเวลา
